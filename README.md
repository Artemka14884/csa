# Arena FPS Online

3D онлайн-шутер в браузере (Three.js + Socket.io) в стиле CS:
- **AK-47** с процедурной 3D-моделью и реалистичным spray-паттерном отдачи (сначала вертикально, затем расхождение вправо/влево, восстановление прицела после прекращения стрельбы).
- **Баннихоп**: физика движения в стиле Source-engine — трение не применяется в момент прыжка, в воздухе разрешено ускорение через strafe (A/D + мышь), что позволяет набирать скорость сериями прыжков.
- **Тренировочная карта**: укрытия для отработки прицеливания, стена-мишень для отработки spray-паттерна AK-47, и отдельная зона с рампами/платформами для баннихопа.
- **Мультиплеер**: сервер на Node.js (Express + Socket.io) синхронизирует позиции игроков, обрабатывает попадания, урон, смерти/респавн и общий чат. Работает с любым числом игроков, у каждого — своё оружие и своя отдача, независимо считаемая на его клиенте.

## Управление
- `WASD` — движение
- Мышь — обзор
- ЛКМ (удерживать) — стрельба очередью, отдача накапливается
- `Пробел` — прыжок (для баннихопа прыгайте сразу в момент касания земли и продолжайте зажимать A/D, слегка покачивая мышью в стороны)
- `R` — перезарядка
- `Shift` — бег по земле
- `T` — чат
- `Tab` — таблица игроков
- `Esc` — отпустить курсор

## Запуск локально

```bash
npm install
npm start
```

Открыть в браузере: http://localhost:3000

Для игры вдвоём и более — откройте эту же ссылку на другом устройстве в той же сети (замените localhost на IP машины), либо задеплойте на render.com (ниже) и делитесь публичной ссылкой.

## Деплой на Render.com

### Вариант 1 — через Blueprint (render.yaml), самый быстрый
1. Залейте эту папку в репозиторий на GitHub (или GitLab).
2. Зайдите на https://dashboard.render.com → **New** → **Blueprint**.
3. Укажите репозиторий — Render сам прочитает `render.yaml` и создаст веб-сервис.
4. Нажмите **Apply**. Через 1–2 минуты сервис будет доступен по адресу вида `https://arena-fps-online.onrender.com`.

### Вариант 2 — вручную
1. **New** → **Web Service** → подключите репозиторий.
2. Настройки:
   - **Runtime**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Plan**: Free (или выше для стабильного онлайна без "засыпания")
3. Deploy.

Сервер сам слушает порт из переменной окружения `PORT`, которую Render передаёт автоматически — ничего дополнительно настраивать не нужно.

## Технические детали отдачи и баннихопа

**Отдача AK-47** (`client.js`, блок `AK_RECOIL_PATTERN`): заранее просчитанный паттерн из 30 патронов — первые выстрелы уводят прицел строго вверх, затем добавляется случайное расхождение вправо-влево с нарастающей амплитудой (имитация реального "спрея" автомата). Смещение накапливается в `recoilOffsetPitch/Yaw` и плавно затухает через `updateRecoilRecovery`, если пауза между выстрелами больше 180 мс.

**Баннихоп** (`client.js`, блок `updateMovement`): при прыжке трение (friction) не применяется к горизонтальной скорости — она полностью переносится в полёт. В воздухе действует ограниченный `wishSpeed`, но ускорение (`accelerate`) не срезает уже набранную скорость, поэтому постоянные strafe-прыжки (A/D + покачивание мыши) позволяют разгоняться выше базовой скорости бега — классическая механика бхопа.
